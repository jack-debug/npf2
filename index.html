npf.2

npf.2 is an updated version of npf, a project I worked on in 2020.

How it works

npf.2 is a packet flooder. It sends multiple packets of information to a specific IP Address in what is essentially a DOS attack.
It uses UDP as the network protocol to send packets.

Improvements from npf

npf.2 uses threads to increase the power of a packet flood. npf.2 also has better performance than npf and has been optimized and the code has been cleaned up.

If you want to try out npf.2, it is publically available on Github.
